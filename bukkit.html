What is Bukkit?

     Ever heard of Minecraft? If not, Minecraft is a multiplayer voxel game developed by the company Mojang in Java. Having multiplayer capabilities a group of people started developing an api that would allow Java developers to more easily modify and control their servers. This group of people turned into the company known as Bukkit and eventually named their api after the same name. Many people in Minecraft's developing community now use this api to add very cool features such as Race Car Driving, Factions (Clans), and Survival games! These mods to the minecraft server are called Bukkit Plugins. I've made some custom plugins for clients I found on Minecraft community websites and I also have a few public plugins on my dev.bukkit.org account. I plan to add more in the future but I mostly make plugins privately.
