What is Unity?

     Unity is amazing and powerful Game Engine featuring Object Oriented Programming, Attribute Scripts, and a 3D world. SpyHop uses this game engine in their Deign Apprenticeship. Unity has been the game engine for many great games such as HearthStone and Monument Valley.


Where can I buy it?
